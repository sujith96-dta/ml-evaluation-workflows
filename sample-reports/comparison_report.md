# OCR Model Evaluation Comparison Report

## Objective

Compare OCR extraction performance between Model V1 and Model V2 using logistics document datasets.

---

# Evaluation Workflow

1. Annotated logistics documents exported from CVAT
2. Ground truth dataset prepared
3. OCR model predictions generated
4. Evaluation pipeline executed
5. Metrics analyzed
6. QA review completed

---

# Summary Results

| Metric | V1 | V2 |
|---|---|---|
| Accuracy | 76% | 100% |
| Precision | 74% | 100% |
| Recall | 73% | 100% |
| F1 Score | 73.5% | 100% |

---

# Key Improvements in V2

- Improved formatting normalization
- Better OCR extraction stability
- Reduced spacing inconsistencies
- Improved logistics entity recognition
- Better invoice number extraction

---

# Annotation QA Impact

Evaluation findings indicate that:
- annotation consistency
- extraction boundary quality
- normalization logic

directly impact OCR model performance.

---

# Final Recommendation

Model V2 demonstrates improved extraction quality and is suitable for expanded validation testing workflows.
